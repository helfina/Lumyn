# Contexte IA — Projet Lumyn

## Présentation

Lumyn est un assistant personnel modulaire conçu pour réduire la charge mentale.

Le projet est développé progressivement en partant de problèmes réels du quotidien. Son premier module permettra de créer rapidement un rendez-vous dans Google Agenda avec des rappels automatiques.

## Identité

- Application : Lumyn
- Moteur intelligent futur : Synapse
- Slogan provisoire : Pensé pour alléger la charge mentale
- Langage : Python
- Framework provisoire : BeeWare, Toga et Briefcase
- Plateformes visées : Windows et Android
- Serveur : aucun serveur prévu pour la première version
